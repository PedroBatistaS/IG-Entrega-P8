# IG-Entrega-P8

Este proyecto muestra cómo cargar un mapa real de Las Palmas de Gran Canaria en un entorno con Three.js y representar encima puntos obtenidos desde un archivo CSV con coordenadas geográficas que representan paradas de taxi en la ciudad de las palmas de gran canaria. Permite visualizar datos reales directamente sobre un plano texturizado como se menciona anteriormente.
